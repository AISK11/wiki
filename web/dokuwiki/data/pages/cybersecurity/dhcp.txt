====== DHCP ======

===== Installation =====

  - Print DHCP version or install it: <cli>
# dhcpcd --version || apt install dhcpcd5
</cli>
  - Uninstall pre-installed obsolete((ISC DHCP website: https://www.isc.org/dhcp.)) ISC dhclient: <cli>
# dhcpcd --version && apt purge isc-dhcp-client isc-dhcp-common
</cli>

===== Configuration =====

  - Set [[https://en.wikipedia.org/wiki/Dynamic_Host_Configuration_Protocol#Options|DHCP options]]: <file text /etc/dhcpcd.conf>
## AUTHOR:       AISK11
## LOCATION:     /etc/dhcpcd.conf (0644 root:root)
## DEPENDENCIES: dhcpcd5
## DESCRIPTION:  DHCP client settings.

## Group to manage dhcpcd via control socket.
controlgroup netdev

## Do not overwrite '/etc/resolv.conf' with DHCP server DNS configuration.
nohook resolv.conf

## Use MAC address as client identifier for DHCP server.
clientid

## Set to blank to avoid sending vendorclassid option.
## Default is "dhcpcd-<version>:<os>:<machine>:<platform>".
vendorclassid

## Request following additional options from DHCP server.
option interface_mtu

## Don't check if obtained IPv4 address is taken by arp (increases speed) and
## also disables IPv4LL (APIPA).
noarp

## Remove any pre-existing IPv4 addresses when adding new IPv4 address.
noalias

## Use MAC address when generating IPv6 SLAAC address for the interface.
slaac hwaddr
</file>
  - Do not start dhcpcd from init: <cli>
# systemctl stop dhcpcd && systemctl disable dhcpcd
</cli>

===== Usage =====

  * Lease IP address for specified interface: <cli>
# dhcpcd <INTERFACE>
</cli>
  * Release IP address for specified interface: <cli>
# dhcpcd -k <INTERFACE>
</cli>
