====== Void Linux ======

{{:logo_voidlinux.svg?200|}}

===== Pros =====

  * SystemD free (runit).
  * Stable rolling release.
  * Independent distribution.
  * AMD64 + ARM support (both 64 and 32).
  * XBPS package manager.
  * Text mode installation.

===== Cons =====

  * Small amount of packages in default repositories.
  * Multiple kernel versions.
