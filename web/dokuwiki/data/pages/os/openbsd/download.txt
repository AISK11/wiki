====== Download ======

1. List of all available FTP mirrors:

  * https://www.openbsd.org/ftp.html#ftp

2. Connect passively and anonymously to the selected FTP mirror host:

<cli>
$ ftp -p <HOST> 21
> anonymous
> anonymous@domain.com 
</cli>

3. Download files into the correct directory and close the connection:

<cli>
ftp> !
$ mkdir -p ~/openbsd/
$ exit
ftp> lcd ~/openbsd/
ftp> cd /pub/OpenBSD/<VERSION>/
ftp> get openbsd-<VERSION>-base.pub
ftp> cd <ARCHITECTURE>/
ftp> get SHA256.sig
ftp> get install<VERSION>.iso
ftp> quit
</cli>

4. Verify downloaded files:

<cli>
$ cd ~/openbsd/
$ signify -C -p *\.pub -x *\.sig *\.iso
</cli>
