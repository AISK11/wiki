====== Gentoo Linux ======

===== Pros =====

  * Rolling release((Versions of various linux distributions: https://en.wikipedia.org/wiki/Comparison_of_Linux_distributions#General.)).
  * Native non-systemd init system (openrc).
  * Independent distribution.
  * Big community and amount of packages((Gentoo packages: https://packages.gentoo.org.)) + pentesting tools((Pentoo: https://pentoo.org.)).
  * Text installation.
  * Extremely minimalistic (CFLAGS).
  * The widest processor architecture support((Download Gentoo: https://www.gentoo.org/downloads.)).
  * Multiple C libraries support (glibc, musl).
  * FSF separation via licenses.
  * Documentation((Gentoo Wiki: https://wiki.gentoo.org/wiki/Main_Page.)).

===== Cons =====

  * Time consuming.
