====== Gentoo Linux ======

===== Pros =====

  * Rolling release((Versions of various linux distributions: https://en.wikipedia.org/wiki/Comparison_of_Linux_distributions#General)).
  * Independent distribution.
  * Native non SystemD init system (OpenRC).
  * Amount of packages in repositories + pentesting tools((Pentoo: https://pentoo.org.)).
  * CLI installation.
  * The widest processor architecture support.
  * Stable.
  * Multiple C libraries support (glibc, musl).

===== Cons =====

  * Source based.
  * Time consuming.

===== Unsure =====

  * Package manager (emerge)?
  * Kernel versions and management?
  * Respecting FSF?
