==== Erase Configuration (WIP) ====

  -Access privileged mode:<cli>
> enable
</cli>
  -If privileged mode is password protected:
    -Physically restart device.
    -During boot phase, hold MODE button until ''switch:'' prompt appears.
    -Delete configuration file and continue booting device::<cli>
switch% flash_init
switch% delete flash:config.text
Are you sure you want to delete "flash:config.text" (y/n)?> y
switch% boot
</cli>
  -Delete device configuration:<cli>
# erase nvram:
# delete vlan.dat
</cli>
  -Verify, that there is no configuration present anymore:
    -Show all file systems:<cli>
# show file systems
File Systems:
     Size(b)     Free(b)      Type  Flags  Prefixes
*   15998976     3329536     flash     rw   flash: flash1:
...
      524288      523212     nvram     rw   nvram:
...
</cli>
    -Show NVRAM memory (3rd row should contain ''0''):<cli>
# dir nvram:
Directory of nvram:/
  510  -rw-  0  no date  startup-config
  511  ----  0  no date  private-config
</cli>
    -Show FLASH memory (''vlan.dat'' should not be present):<cli>
# dir flash:
</cli>
  -Restart device to apply any unsaved changes:<cli>
# reload
</cli>

==== Erase Configuration ====

  -Access privileged mode:<cli>
> enable
</cli>
  -If privileged mode is password protected:
    -Physically restart device.
    -During boot phase, press <kbd>break</kbd> (type ''~#'' if using cu command) or try <kbd>ctrl</kbd>+<kbd>c</kbd>.
    -Set configuration register((Cisco router configuration register values: https://www.cisco.com/c/en/us/support/docs/routers/10000-series-routers/50421-config-register-use.html#anc6.)) to use NVRAM configuration (''0x2142'') and restart device to apply changes:<cli>
rommon> confreg 0x2142
rommon> reset
</cli>
  -Delete device configuration:<cli>
# erase nvram:
# delete vlan.dat
</cli>
  -Verify, that there is no configuration present anymore:
    -Show all file systems:<cli>
# show file systems
File Systems:

       Size(b)     Free(b)      Type  Flags  Prefixes
...
        262136      254916     nvram     rw   nvram:
...
*    255537152   111308800  usbflash     rw   usbflash0: flash:
</cli>
    -Show NVRAM memory (3rd row should contain ''0''):<cli>
# dir nvram:
Directory of nvram:/

  254  -rw-  0  no date  startup-config
  255  ----  0  no date  private-config
  256  -rw-  0  no date  underlying-config
...
</cli>
    -Show FLASH memory (''vlan.dat'' should not be present):<cli>
# dir flash:
</cli>
  -Check if correct configuration register (''0x2102'') is used:<cli>
# show version | i register
Configuration register is 0x2102
</cli>
  -If there is different configuration register:
    -Restart device:<cli>
# reload
</cli>
    -During boot phase, press <kbd>break</kbd> (type ''~#'' if using cu command) or try <kbd>ctrl</kbd>+<kbd>c</kbd>.
    -Set configuration register to use NVRAM (configuration) and restart device to apply changes:<cli>
rommon> confreg 0x2102
rommon> reset
</cli>
  -Restart device to apply any unsaved changes:<cli>
# reload
</cli>
