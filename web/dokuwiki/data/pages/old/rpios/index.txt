====== Raspberry Pi OS Linux ======

===== Pros =====

  * Focused and optimized for Raspberry Pi single-board computers.
  * Stable.
  * Binary based.
  * Friendly website.

===== Cons =====

  * SystemD.
  * Point release
  * Package manager.
  * Depends on Debian Linux.
  * Mostly pre-installed (harder to modify).
